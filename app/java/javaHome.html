1. Removing all duplicates from a list in Java8

Using Lambdas in Jav8, We are going to use the distinct() method from Stream API to remove duplicates from list 
import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

public class RemoveDuplicatesFromList {

	public static void main(String args[]){
		List duplicateList = Arrays.asList(1,2,1,3,2);
		List distinctList = duplicateList.stream().distinct().collect(Collectors.toList());
		System.out.println(distinctList);
	}
}							    					
			    
Output: [1, 2, 3]

2. Sort list of objects with a specific property in Java8

In Jav8, we can use below code to sort the object 
package com.vijay.java8;

import java.util.ArrayList;
import java.util.Comparator;
import java.util.List;

public class RemoveDuplicatesFromList {

	public static void main(String args[]){
		Customer customer1 = new Customer();
		customer1.setId(2);
		customer1.setName("Vijay");
		Customer customer2 = new Customer();
		customer2.setId(1);
		customer2.setName("Bhaskar");
		List customerList = new ArrayList();
		customerList.add(customer1);customerList.add(customer2);
		customerList.forEach(customer->System.out.println("Before Sort: "+customer.getId()+","+customer.getName()));
		//Sort
		customerList.sort(Comparator.comparing(Customer::getId));
		
		customerList.forEach(customer->System.out.println("After Sort: "+customer.getId()+","+customer.getName()));
	}
}

Customer: This is a POJO(Plain Old Java Object) object which has id and name properties with setters and getters.						    					
			    
Output: Before Sort: 2,Vijay
Before Sort: 1,Bhaskar
After Sort: 1,Bhaskar
After Sort: 2,Vijay
